| Поле | Описание |

| `category_id` | Уникальный идентификатор категории |

### `products`

|---|---|

|---|---|

## Статусы заказов

| Статус | Значение |
|---|---|
| `created` | Заказ создан, но еще не оплачен |
| `paid` | Заказ оплачен |
| `shipped` | Заказ передан в доставку |
| `delivered` | Заказ доставлен клиенту |
| `cancelled` | Заказ отменен |
| `returned` | Заказ возвращен |

## Статусы оплат

| Статус | Значение |
|---|---|
| `pending` | Оплата ожидается |
| `paid` | Оплата успешно выполнена |
| `failed` | Оплата не прошла |
| `refunded` | Деньги возвращены клиенту |

## Статусы доставок

| Статус | Значение |
|---|---|
| `not_started` | Доставка еще не началась |
| `in_progress` | Заказ находится в доставке |
| `delivered` | Заказ доставлен |
| `failed` | Доставка не выполнена |
| `returned` | Заказ возвращен |

## Расчетные показатели

| Показатель | Формула | Назначение |
|---|---|---|
| `revenue` | `quantity * unit_price * (1 - discount_percent / 100)` | Выручка с учетом скидки |
| `total_cost` | `quantity * cost` | Себестоимость проданных единиц |
| `gross_profit` | `revenue - total_cost` | Валовая прибыль |
| `gross_margin_percent` | `gross_profit / revenue * 100` | Валовая маржа, % |
| `average_order_value` | `revenue / orders_count` | Средний чек |
| `revenue_per_customer` | `revenue / customers_count` | Выручка на клиента |
| `delivery_delay` | `actual_delivery_date - planned_delivery_date` | Задержка доставки в днях |
| `recency` | `analysis_date - last_order_date` | Давность последнего заказа |
| `frequency` | `count(distinct order_id)` | Частота покупок |
| `monetary` | `sum(revenue)` | Денежная ценность клиента |
| `ABC group` | Группа по накопительной доле выручки | Приоритизация товаров |

## ABC-группы

| Группа | Интерпретация |
|---|---|
| `A` | Товары, формирующие основную долю выручки |
| `B` | Товары со средним вкладом |
| `C` | Товары с небольшим вкладом в выручку |

## Примечание по формулам

Для расчета финансовых метрик рекомендуется учитывать только релевантные заказы: например, `paid`, `shipped`, `delivered` в зависимости от аналитической постановки. Возвраты и отмены нужно анализировать отдельно или исключать из основной выручки.
