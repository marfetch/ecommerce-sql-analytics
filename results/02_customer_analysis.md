# Анализ клиентов

## Цель анализа

Оценить структуру клиентской базы: повторные покупки, ценность клиентов, RFM-сегменты, распределение по городам и вклад клиентских групп в выручку.

## Используемые запросы

Раздел использует запросы из `sql/08_analytics_queries.sql`:

- повторные покупки клиентов;
- RFM-сегментация клиентов;
- рейтинг городов по выручке;
- эффективность каналов привлечения;
- агрегаты из `v_customer_orders`.

## Повторные покупки

### Метрики

- количество клиентов;
- количество клиентов с одним заказом;
- количество клиентов с повторными заказами;
- доля repeat customers;
- средняя выручка на клиента.

| Метрика | Значение |
|---|---:|
| Всего клиентов | 634 |
| Клиенты с 1 заказом | 136 |
| Клиенты с 2+ заказами | 498 |
| Доля repeat customers, % | 78,55 |
| Выручка на клиента | 208 298,25 |

### Интерпретация

Высокая доля повторных покупок говорит о хорошем удержании и ценности клиентской базы. Низкая доля может означать проблемы с повторной коммуникацией, ассортиментом, доставкой или клиентским опытом.

### Бизнес-вывод

Города с высокой выручкой и большим числом повторных покупок можно рассматривать как приоритетные для маркетинга и логистики. Города с большим количеством клиентов, но низкой выручкой требуют дополнительной проверки среднего чека и ассортимента.

## Возможные выводы

- Основная выручка может концентрироваться в небольшой группе лояльных клиентов.
- Большая доля one-time клиентов указывает на потенциал CRM-коммуникаций.
- RFM-сегментация помогает различать удержание, реактивацию и VIP-подход.
